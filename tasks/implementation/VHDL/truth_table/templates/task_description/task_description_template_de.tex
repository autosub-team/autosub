\section*{Truth Table}

Ihre Aufgabe ist es, das Verhalten einer Entity  namens "`truth\_table"' (Wahrheitstabelle) zu programmieren. Die Schnittstelle ist in der angeh\"angten Datei "`truth\_table.vhdl"' deklariert und hat folgende Eigenschaften:
\begin{itemize}
\item Eing\"ange:  A, B, C, D vom Typ  std\_logic
\item Ausgang: O vom Typ std\_logic
\end{itemize}
